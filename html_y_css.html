1. HTML
    • What is HTML?
      Hypertext Markup Language

    • Explain the head and the body tags.
      The head is where it contains data about the page.
      The body is what is going to show in the page.

    • Explain the section, article, aside, header,footer tag.
      1. The section tag is used to group together related elements.
      2. The article tag specifies independent, self.contained content.
      3. The aside tag marks up content that is indirectly related to the main article content of a page.
      4. The header tag specifies a header for a document or section.
      5. The footer tag defines a footer for a document or section.

  2. CSS
    • What is CSS
      Cascading Style Sheets

    • What is the purpose of CSS
      CSS describes the presentation of a document written in HTML.

    • How do you apply a CSS to an HTML
      1. You can implement inline, which it is apply to a single element.
      2. Using the tag 'style' in the 'head' section of the HTML document.
      3. Creating a separate .css file and linking it to the HTML document.

    • What is a Selector? What is a CSS rule
      A selector is a pattern of elements and other ter terms that tell the browser which HTML elements should be selected to have the CSS property values.
      A rule is a statement that provides CSS with instructions to perform or how to behave.
    • Explain a:
      • Tag Selector : used to select the HTML elements you want to style.
      • Class Selector : selects all elements that have the given class attribute denoted by the class name prefixed with a period.
      • Tag and Class Selector :selects all tags that are realeted with a class specified.
      • Id Selector : matches an element based on the value of the element's id attribute.

    • What is a class? What is an id? How are they related to CSS?
      A class is a group of elements that are the same or similar.
      An id is a identifier that can be assigned to an element on a webpage.
      Both classes and Ids are used to identified or select a target and style HTML elements.

    • Explain the CSS Box Model? Show an example.
      The CSS Box Model is a container that contains multiple properties including borders, margin, padding, and the content itself.

    • How can you change the font color using CSS?
      Add the appropriate CSS selector and define the color property with the value you want.
